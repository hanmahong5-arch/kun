# 施工企业市场经营管理小程序 - 功能实现说明

## 已实现功能清单

### 1. 周报填报功能 ✅
**页面路径**: `/pages/reports/create/index`

**功能特性**:
- 支持填写本周核心工作、项目跟踪、投标工作、客户对接、下周计划等必填项
- 支持填写存在问题与协调需求（可选）
- 自动计算当前周的开始和结束日期
- 支持保存草稿和正式提交两种方式
- 必填项校验，确保数据完整性
- 提交后状态变更为"待审阅"

**使用方式**:
- 市场经营人员可通过首页快捷入口或工作汇报模块进入
- 填写完成后可选择"保存草稿"或"提交周报"

---

### 2. 周报查阅功能（领导专属） ✅
**页面路径**: 
- 审阅列表: `/pages/reports/review/index`
- 周报详情: `/pages/reports/detail/index`

**功能特性**:
- 仅领导和系统管理员可访问
- 支持按状态筛选：待审阅、已通过、已驳回、全部
- 显示填报人、所属部门、填报周期、提交时间等信息
- 支持查看周报详细内容
- 支持添加审阅意见
- 支持通过或驳回操作
- 驳回后员工可重新编辑提交

**使用方式**:
- 领导通过首页快捷入口"周报审阅"进入
- 点击周报卡片查看详情
- 在详情页填写审阅意见并选择通过或驳回

---

### 3. 项目填报功能 ✅
**页面路径**:
- 新增项目: `/pages/projects/create/index`
- 编辑项目: `/pages/projects/edit/index`

**功能特性**:
- 支持填写项目名称、项目分级、建设单位、工程类型等必填项
- 支持填写投资规模、项目阶段等可选项
- 项目分级支持：A锁项目、A争项目、B类项目、C类项目、D类项目
- 自动关联当前登录用户为对接负责人
- 支持后续编辑项目信息

**使用方式**:
- 通过首页快捷入口"新增项目"进入
- 或在项目管理列表页点击"新增"按钮
- 填写完成后点击"提交"保存

---

### 4. 任务指派功能（领导专属） ✅
**页面路径**: `/pages/tasks/assign/index`

**功能特性**:
- 仅领导和系统管理员可访问
- 支持填写任务名称、任务类型、任务描述等必填项
- 支持选择责任人（从已审核通过的用户中选择）
- 支持设置优先级：低、中、高
- 支持设置截止时间
- 支持关联项目或客户（可选）
- 任务创建后自动通知责任人

**使用方式**:
- 领导通过首页快捷入口"任务指派"进入
- 填写任务信息并选择责任人
- 设置优先级和截止时间
- 点击"指派任务"完成

---

### 5. 年度经营目标统计展示（领导专属） ✅
**位置**: 首页底部

**功能特性**:
- 仅领导和系统管理员可见
- 显示2026年度目标完成情况
- 总体完成率进度条可视化展示
- 各类项目数量统计（A类、B类、C类、D类）
- 累计投资规模统计
- 数据实时更新

**数据来源**:
- 从projects表统计各类项目数量
- 计算投资规模总和
- 根据A锁项目占比计算完成率

---

### 6. 数据导出功能（领导专属） ✅
**页面路径**: `/pages/reports/export/index`

**功能特性**:
- 仅领导和系统管理员可访问
- 支持三种数据导出类型：
  1. **周报数据**: 按月导出，包含填报人、部门、周期、各字段内容、审阅状态等
  2. **项目数据**: 按分类导出（全部/A锁/A争/B类/C类/D类），包含项目名称、分级、建设单位、工程类型、投资规模、负责人等
  3. **客户数据**: 按类型导出（全部/政府单位/国有企业/民营企业/其他），包含客户名称、类型、分级、联系方式等完整信息
- 导出格式为Excel文件（.xlsx）
- 文件名自动包含导出类型和日期

**使用方式**:
- 领导通过首页快捷入口"数据导出"进入
- 选择导出类型
- 设置筛选条件（月份/分类/类型）
- 点击"开始导出"生成Excel文件

---

### 7. 首页快捷入口 ✅
**位置**: 首页待办事项卡片下方

**功能入口**:
- **所有用户可见**:

  - 新增项目：快速进入项目新增页面

- **领导专属**:
  - 任务指派：快速进入任务指派页面
  - 周报审阅：快速进入周报审阅列表
  - 数据导出：快速进入数据导出页面

## 数据库表

### weekly_reports（周报表）
- 存储用户提交的周报数据
- 包含必填字段和可选字段
- 支持草稿和提交状态
- 记录审阅信息

### projects（项目表）
- 存储项目基本信息
- 支持项目分级管理
- 关联负责人信息

- 存储任务指派信息

- 关联项目和客户

## 技术实现

### 前端技术栈
- Taro + React + TypeScript
- Tailwind CSS样式
- 响应式设计

### 后端技术栈
- Supabase数据库
- Row Level Security (RLS) 权限控制
- 实时数据同步

### 数据导出
- xlsx库生成Excel文件
- 支持多种筛选条件
- 自动格式化数据
